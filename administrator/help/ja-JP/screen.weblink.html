Web Links Manager

-- Draft 4.5.3

Web Links Manager はウェブリンクメニューアイテムを通してユーザーに提案するウェブサイトへのリンク一覧を表示します。

ツールバーアイコン

それぞれの列の先頭にあるチェックボックスをクリックしてウェブリンクを選択します。

公開と非公開: : 選択されたいくつかのウェブリンクを公開/非公開にするためには、’公開’あるいは’非公開’アイコンをクリックします。複数のチェックボックスにチェックできます。

削除: 選択したウェブリンクを削除します。この選択で良いか確認させるために警告がポップアップで出てきます。

編集 : 存在するウェブリンクパラメータを編集します。

新規 : 新しいウェブリンクを作成します。

フィルター

左から右へ:
名前でウェブリンクを検索します。
状況: 公開/非公開の状態を表示します。
カテゴリ: ウェブリンクが属するカテゴリを表示します。

コラム説明

タイトルのクリックでクリックしたタイトルによる表示順に並び替えられます。

チェックボックス/南京錠: アイテムを選択するにはこれをクリックします。アイテムがチェックアウトされた場合は、南京錠アイコンがチェックボックスの代わりに表示されます。個人個人でそれをクリックしてアイテムにチェックインします。もしこのアイテムが他のマネージャー/管理者によってチェックアウトされた場合は、あなたのサイトの管理者（彼/彼女）にアイテムにチェックインするよう連絡を取ってください。チェックされれば、ツールバーアイコンをクリックして利用できる様々な設定を実行できます。
 
タイトル: これはウェブリンクの名前です。名前をクリックしてウェブリンクを編集します。
 
カテゴリ: これはウェブリンクが分類されているカテゴリーです。クリックしてカテゴリを編集します。
 
公開: ウェブリンクを公開するかどうかを決めます。状態を変更するにはアイコンをクリックします。

並替 : ウェブリンク一覧の並び順を表示します。アイテムの表示順序を上や下に並べ替えたいときは矢印アイコンをクリックします。

順序 : ウェブリンク一覧の順序を表示します。並んでいる数字を表示したい順に編集し、列見出しにある‘順序’の隣のフロッピーマークアイコンをクリックして一覧のウェブリンクの順序を変更できます。

ヒット: ウェブリンクが参照された回数です。

ID: データベースに保存されたウェブリンクのIDです。
 
一覧の一番下にページ付け機能がありますが、これはあなたを次の一覧アイテムへのナビゲートをしています。また画面下のドロップダウンボックスにある数字を選択して１ページに表示するアイテム数を設定できます。
