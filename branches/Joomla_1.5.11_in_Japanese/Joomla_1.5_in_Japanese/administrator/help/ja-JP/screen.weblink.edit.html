Web Link Manager : 新規 / 編集

-- Draft 4.5.3

Edit Web Link ページではウェブリンクの詳細を追加編集できます。

ツールバーアイコン

保存: 作成/編集したウェブリンクを保存し、ウェブリンクマネージャーに戻ります。

適用: 作成/編集したウェブリンクを保存し、編集画面を保持します。

閉じる: 実行をキャンセルします。いかなる編集も失われます。編集するために開いたなら、ウェブリンクはチェックイン状態となります。

詳細

名前: ここにあなたのサイトに表示されるウェブリンクの名前を入力します。
 
カテゴリ: 分類するカテゴリを選択します。
 
URL: ウェブリンクのURLを入力します。
 
説明: ウェブリンクの説明テキストを付け加えます。
 
表示順: ここで特定のカテゴリ内でのウェブリンクの表示順を選択します。ウェブリンクカテゴリメニューアイテムをテーブル形式で表示したときにその順番に表示されます。
 
承認: 作成されたウェブリンクを管理者の承認を得てからの公開にするかどうかをここで決定します。
 
公開: ウェブリンクを公開するかしないかを選択します。

パラメータ

ターゲット: リンクをクリックした時に同じウィンドウ（親ウィンドウ+ブラウザナビゲーション）で開く、新規ウィンドウ+ブラウザナビゲーションで開く、あるいは新規ウィンドウ+ブラウザナビゲーションなしで開く、のどれかを決定します。
